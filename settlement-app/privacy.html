経費精算 for BEATINK (Settlement) — iOS アプリ

1. 概要

本アプリ「経費精算 for BEATINK」（以下「本アプリ」）は、株式会社ビートインクの社員が領収書・電車交通費・タクシー交通費を撮影／入力し、Google Sheets 上の精算書に自動転記するために作成された社内向けアプリです。開発・提供は個人（XTAL-JP）が行っており、一般公開および商用配信は想定していません。

2. 収集する情報

本アプリの利用に伴い、以下の情報を各外部サービスへ送信します。本アプリ独自のサーバーは保有せず、開発者側のデータベースに情報を蓄積することはありません。

  • アカウント情報: Google アカウントのメールアドレスおよび Google Sheets への書き込み権限を取得（Google Sign-In OAuth 経由）
  • 領収書画像および OCR 抽出情報: ユーザーが撮影した領収書画像を Anthropic Claude API に送信し、日付・店舗名・金額・タクシー判定を抽出します。画像は処理後に破棄されます
  • 駅名: 電車運賃を取得するため、出発駅と到着駅の名称を Brave Search API および Anthropic Claude API に送信します
  • 精算データ: 日付・金額・摘要・プロジェクト名・会社名（BEAT / BEGGARS）等をユーザー自身の Google Spreadsheet に書き込みます

広告 SDK・アクセス解析ツール（Google Analytics 等）・クラッシュレポート SDK は組み込んでいません。

3. 情報の利用目的

  • 撮影した領収書からの情報自動抽出（OCR）
  • 電車運賃の自動検索
  • 抽出結果および入力データのユーザー Google Sheets への書き込み

4. 情報の保存場所

本アプリはユーザーデータを開発者側のサーバーに保存しません。以下の外部サービスに送信された情報は、それぞれの事業者のプライバシーポリシーに従って取り扱われます。

  • Google Sheets / Google Sign-In（Google プライバシーポリシー）
  • Anthropic Claude API（Anthropic Privacy Policy）
  • Brave Search API（Brave Search Privacy）

精算データそのものはユーザーが指定した Google Spreadsheet 上に保存され、開発者はその内容にアクセスしません。

5. 第三者への提供

本アプリは、ユーザーが登録したデータを第三者に販売・提供しません。第 4 項に列挙した外部 API サービスへの送信は、機能提供のために必要な範囲で行われます。

6. データの削除

Google Sheets に書き込まれたデータは、ユーザー自身が Google Sheets 上で削除できます。Google アカウントのアプリ連携解除は、ユーザーの Google アカウント設定画面から実施できます。その他のご相談は下記お問い合わせ先までご連絡ください。

7. 子供の利用について

本アプリは業務目的の社内向けアプリであり、13 歳未満の子供の利用は想定していません。子供に関する情報を意図的に収集することはありません。

Privacy Policy

Settlement for BEATINK — iOS App

1. Overview

"Settlement for BEATINK" (the "App") is an internal iOS application built to help employees of BEATINK Inc. capture receipts and railway/taxi transportation expenses and post them automatically to a Google Sheets settlement worksheet. The App is developed and operated by an individual (XTAL-JP) and is not intended for the general public or commercial distribution.

2. Data We Collect

The App does not operate its own server and does not persist any user data on the developer's side. During normal use the App transmits the following information to third-party services:

  • Account information: Google account email address and Google Sheets write permission (via Google Sign-In OAuth)
  • Receipt images and OCR results: Photographs taken by the user are transmitted to the Anthropic Claude API to extract date, store name, amount, and taxi flag. Images are discarded after processing
  • Station names: Departure and arrival station names are sent to the Brave Search API and the Anthropic Claude API to look up train fares
  • Settlement records: Date, amount, description, project name, company label (BEAT / BEGGARS), etc. are written to the user's own Google Spreadsheet

The App does not include any advertising SDKs, analytics tools (such as Google Analytics), or crash reporting SDKs.

3. Purpose

  • Automatic information extraction (OCR) from receipt photographs
  • Automatic train fare lookup
  • Writing extracted and manually entered data to the user's own Google Sheets

4. Where Data Is Stored

The App does not store user data on developer-controlled servers. Information transmitted to the following third-party services is handled according to each provider's respective privacy policy:

  • Google Sheets / Google Sign-In (Google Privacy Policy)
  • Anthropic Claude API (Anthropic Privacy Policy)
  • Brave Search API (Brave Search Privacy)

Settlement records themselves are stored in the Google Spreadsheet designated by the user. The developer does not access the contents of these records.

5. Third-Party Sharing

We do not sell, rent, or share user data with third parties. Transmissions to the external APIs listed in section 4 are limited to what is necessary to provide the App's features.

6. Data Deletion

Data written to Google Sheets can be deleted directly by the user in Google Sheets. The App's authorization can be revoked from the user's Google account settings. For any other inquiries, please contact us using the address below.

7. Children

This App is a business-oriented internal tool. Use by children under 13 is not envisioned, and we do not knowingly collect information from children.
